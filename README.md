# Workout-App
Workout-App 

Date: 3/05/2021 

By: Nick Williams 








# Description
The Workout-App is an App that people pick the type of workout they want to do. They can add whatever workout they want to do in that category, and also remove it to.   


# Technologies 
* Express.js

* MongoDB

* CSS

* React 

* Node.js
